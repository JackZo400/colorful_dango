## ✨ Features

- End-to-end encrypted chat
- Message recall & clear (both sides)
- Contact persistence
- Dark/light theme
- Bilingual UI (Chinese/English)
- Colorful title · Splash animation
- Self-hosted signaling server (cross-network)

## 🌐 Self-hosted Signaling

```bash
scp bin/server.dart root@your-vps:/root/
ssh root@your-vps && apt-get install -y dart
nohup dart run /root/server.dart > /var/log/dango.log 2>&1 &
# Connect to ws://your-vps:8765
```

## 🛠 Build
